# 地动

## 基本属性

- 法术类型：防御
- 法术流派：磐石
- 魔力需求：12
- 法术稀有度：普通
- 法术强度倍率：0.2
- 法术ID：earth_tremor
- 英文：Earth Tremor Spell
- 释放类型：本体
- 可见性：可见

## 描述
地动法术会在施法者周围产生一次小范围的地震，地震会将除玩家外范围内的所有生物击退至地震范围外，并造成少量物理伤害。

## 数值

### 基础数值

击退伤害：*2*  
地震范围：*5 * 5*（不可以被修正）  
击退范围：*8 * 8*（不可以被修正）  

| 属性      | 数值    | 能否修正 |
| ----------- | ----------- |  :---: |
| 击退伤害      | 2      | 是 |
| 地震范围      | 5 * 5       | 否 |
| 击退范围   | 8 * 8        | 否 |

### 流派属性
**沉重**：击退效果造成的伤害无视单位的盔甲韧性。


## 法术精进

- 熟练：击退伤害提高至 *3*
    - 精进需求：  
- 掌握：地震范围提高至 *6 * 6*，击退范围提高至 *10 * 10*
    - 精进需求：  
- 共鸣：击退伤害提高至 *4*  
    - 精进需求：  
- 谐律：被法术击退的目标额外获得**眩晕**效果，持续2秒
    - 精进需求：
