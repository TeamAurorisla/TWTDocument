# 火盾

## 基本属性

- 法术类型：防御
- 法术流派：炽焰
- 魔力需求：8
- 法术稀有度：普通
- 法术强度倍率：0.5
- 法术ID：fire_shield
- 英文：Fire Shield Spell
- 释放类型：本体
- 可见性：可见

## 描述
火盾法术会在施法者周围产生一个炽热的火焰护盾，护盾可以承受一定次数或一定量的伤害，当护盾被击破时，会对周围敌对单位造成火焰伤害。

## 数值

### 基础数值

护盾承受伤害次数：*4*（不可以被修正）  
护盾承受伤害总量：*15*（不可以被修正）  
护盾击破伤害：*6*  
护盾持续时间：*30s*（不可以被修正）

| 属性      | 数值    | 能否修正 |
| ----------- | ----------- |  :---: |
| 护盾承受伤害次数      | 4      | 否 |
| 护盾承受伤害总量      | 15       | 否 |
| 护盾击破伤害   | 6        | 是 |
| 护盾持续时间   | 30s        | 否 |

### 流派属性数值

**灼烧**：当护盾被击破时，有30%的概率对周围敌对单位附加**灼烧**效果，持续造成火焰伤害。  

**畏水**：当玩家处于水中时，护盾直接失效。当玩家处于露天场景且下雨时，视为每2.5秒受到1点伤害。  

## 法术精进

- 熟练：护盾承受伤害次数提高**1次**，护盾击破伤害提高**2点**，灼烧效果概率提高至**40%**  
    - 精进需求：
- 掌握：护盾承受伤害总量提高**5点**，灼烧效果概率提高至**60%** 
    - 精进需求：
- 共鸣：护盾承受伤害次数提高**1次**，护盾承受伤害总量提高**5点**，灼烧效果概率提高至**80%**
    - 精进需求：
- 谐律：灼烧效果概率提高至**100%**，当护盾被击破时，额外给予玩家一个**魔力充盈**一级效果，持续**15秒**
    - 精进需求：
