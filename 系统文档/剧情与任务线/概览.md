# 概览


## 任务的分类

| 类别 | 名称 | 名称 | 说明 | 示例 |
| :--- | :---: | :---: |:---| :--- |
| 主线任务 | 征途 | Odyssey | 有着连贯流程和剧情的完整旅程，以“探秘失落遗迹”为麦高芬，按照学派分成不同的章节，每一个章节较长且内容丰富 | 见主线 |
| 剧情支线 | 见闻 | Tales | 有完整故事线，以流派或玩法作为主题，目标是推动玩家发展；会在笔记里有引导，玩家可以主动接取或在探索到关键结构时触发 | 雪国遗章 [ 发现古老的城堡里有着奇怪的动静... ] ：寻找寒冰之下的古老城堡 -> 跟踪袭击者 -> 击败冰霜傀儡，获得寒冰符文|
| 探索支线 | 游历 | Encounters | 探索途中偶然触发的小事件，轻量化任务或小故事，增加世界生动性；只能被动触发，在完成后会记录在笔记中 | 遇见遗迹守卫的残影，协助完成誓言 |
| 图鉴支线 | 图鉴 | Records | 无剧情，将生物和结构以图鉴的形式展示出来，方便玩家寻找；以收集材料为主 | 按照类别分类的建筑与生物 |
