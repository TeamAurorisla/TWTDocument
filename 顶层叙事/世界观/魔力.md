# 魔力

魔力来自于本源，是一种特殊的能量。人们并没有真正掌握魔力，而是通过不同的流派作为管道来利用魔力。魔法使们通过学习与实践，掌握了不同流派的魔法知识与技能，从而能够操纵魔力，施展各种魔法效果。当然，在未来，对于魔力的运用会更加深入。

> 原有设定，符合世界观设定，但需要大幅度修改  
>### 魔法革命
>《标准魔法论》的诞生，标志着标准化、系统化的元素魔法理论正式确立，形成了炽焰、寒冰、雷电、疾风、磐石五大流派，取代了中世纪依赖个人天赋、零散混乱的魔法知识与实践。  
>- 魔法教育门槛高昂，魔法教育资源被精英阶层垄断，形成了魔法阶层垄断，社会地位迅速提升，与贵族阶层相当。  
>- 魔法使推崇探索精神，发起“魔法文艺复兴”，主张通过实践与冒险推动魔法与力量的进步.
>- 他们对其他文明与失落遗迹中的魔法知识抱有浓厚兴趣，积极投身于魔法的研究与探索。  
>### 宗教改革
>当时，王国旧教廷认为一切力量源于神明，操纵标准化的元素魔法被视为僭越神权。为此，教廷设立了神圣裁判所，对魔法使进行迫害与审判。  
>魔法使们协助王权实现了复兴运动，因而获得王室支持。随后，王国爆发了宗教改革，诞生了新教，主张信仰自由与个人与神明的直接联系，反对教会对于神秘的垄断。他们将教会原有的神圣术法体系化，列为与五大流派并列的第六流派——神圣流派  
>### 反魔法运动
>魔法使建立了规模巨大的实验设施与魔法工房进行魔法实现，并雇佣当地居民作为助手与劳工，导致没有受到系统魔法教育的平民劳工受到了魔力的巨大伤害。因此，雇工们自发的组织起了反魔法运动，通过破坏实验设施与暴力抗争来反对魔法使的压迫和剥削，要求取缔魔法实验。  
在压力之下，魔法使作出了让步。“大魔法使协会”主导创造了第七流派契约流派。其核心是召唤并绑定异界魔物或元素生命，让它们承担危险的任务。同时，《王国魔法实验与劳动伦理规范法》等一系列被强制推行。
