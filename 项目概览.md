# 项目概览

「**航迹：潮汐绘卷 The Weaving Tides**」是一个大型剧情向海岛魔法冒险包。玩家将扮演一位大航海时代的探险家，征服广袤的海洋，收集宝藏与遗珍，并最终揭开一个失落文明的秘密。

**故事背景**：文艺复兴时期，大航海时代；玩家作为新晋冒险家，受到王国特许公司的一个海岛总督的雇佣，成为一个探险家，寻找埋藏在海洋中的宝藏，探索古老的秘密。

**核心游戏流程**：寻找宝藏 -> 玩家发展（法术与构筑系统） -> 拓展舰队（基地，非核心模块）-> 寻找宝藏

--- 

### 主线剧情：
- 玩家将会出生在总督的首都贸易港，完成新手教程，引入世界观
1. 剿灭周边海盗，熟悉基础操作与战斗系统
2. 拓展海图，寻找指定宝藏（主要游戏区间）
3. 挑战失落文明遗迹，发现遗珍（中后期游戏主线）
4. 集齐所有遗珍，挑战终极遗迹，击败最终BOSS，成为公司新任总督（游戏目标）
- 日常任务：接取日常任务，获得基础资源

### 核心特色：
- 偏向ARPG的策略向战斗系统，独特的原创魔法模组与玩家发展体系。
- 丰富且成体系的冒险和探索内容，原创的地图和生物。
- 贸易岛初始地图：包含首都府邸、集市（交易常用材料和宝藏）、港口等，玩家可以在20分钟内探索完毕
- 剧情向的引导系统和NPC系统，方便引导和沉浸感。
- 减少时间浪费的航线系统：在海洋上，玩家可以沿着航线快速移动至已知贸易港和海岸。贸易港是航线的节点，在传送过程中，有几率触发随机事件（小游戏），增加航海的趣味性和不确定性。
- 大航海时代风格的任务日志系统

### 冒险与探索
- 宝藏分为两种：宝藏和遗珍。宝藏会在战利品箱或Boss战中生成，根据稀有程度分为普通宝藏和史诗宝藏。宝藏往往可以使用或佩戴，给玩家带来少量增益或技能。遗珍会且只会在远古遗迹中完成挑战获得，与失落文明的故事息息相关。只要获得即解锁永久性祝福。
- 结构是玩家游戏的核心，普通结构包含少量战利品，核心结构规模更大，是游戏的主线。结构主题可以和岛屿，海岛，海底有关，结构的挑战可以更加多样，包括但不限于Boss rush，箱庭世界，解谜；特别的是，下界重塑为一个有有进入次数或时间限制的特殊空间，可以获得特殊材料，末地则是有着一个大型Boss关的核心结构。
- 远古遗迹是与失落文明有关的结构，是游戏中后期的主线，只能在藏宝图中发现，在寻找远古遗迹时可以添加一些小游戏进行解密（类似冬季救援）。

### 法术与战斗系统

**航迹：潮汐绘卷**的战斗系统以独特的原创魔法模组为核心，强调策略深度与玩法多样性，致力于提供富有思考性的战斗体验，而非依赖快速反应或固定的技能循环。法术系统受《部落冲突：皇室战争》等游戏的启发，玩家需从已解锁的法术中选取8个（不可重复）装备于**魔法槽**中。战斗中，玩家可从随机出现在候选槽中的4个法术里选择施放；法术释放后，系统会从法术槽内不与当前候选槽重复的法术中随机抽取一个进行填充，从而形成动态、可适应的战术选择。

法术以七大元素**流派**为基础：炽焰、寒冰、雷电、疾风、磐石、神圣、契约，每个流派具备独特的视觉效果与玩法特色。为了进一步深化策略维度，游戏引入了**稳态魔力**与**涌态魔力**两种魔力：稳态魔力随时间自然恢复，而涌态魔力需通过攻击或受击获取。此外，玩家还可采取**魔力超载**机制——当施法消耗超出当前魔力值但未超过（当前魔力+4）时，仍可强制施法，并触发超载效果，使魔力上限临时降低4点。在超载状态下，玩家无法连续触发超载，以此平衡风险与收益。

法杖采用**模块化**设计，玩家可通过搭配不同模块来自定义法杖性能，以适应各类战术需求。法术本身具备丰富的发展维度，包括精进、刻印，并存在独特的传奇法术与能力技能；法杖也可进行特化与宝石镶嵌，共同构建深度的自定义空间。

此外，玩家成长体系的核心在于**构筑系统**。该系统采用类似《群星》的模块化树状结构，玩家可在构筑树中选择不同的独立模块，通过“点亮节点”逐步提升角色能力。当一个模块的全部节点被点亮后，将解锁一项飞升天赋，带来机制性突破或显著强度提升。
