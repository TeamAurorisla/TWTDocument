# 火球术

## 基本属性
- 法术类型：攻击
- 法术流派：炽焰
- 魔力需求：6
- 法术稀有度：普通
- 法术强度倍率：0.5
- 法术ID：fireball
- 英文：Fireball Spell
- 释放类型：跟踪
- 可见性：可见

## 描述
火球术会释放出一个巨大的火球，在击中目标时爆炸，造成大量伤害并附有小范围的爆炸效果和范围内的火焰燃烧效果。

## 数值
### 基础数值
火球伤害：*16*  
火球爆炸范围：*2 * 2 * 2*（不可以被修正）  

| 属性      | 数值    | 能否修正 |
| ----------- | ----------- |  :---: |
| 火球伤害     | 16      | 是 |
| 火球爆炸范围      | 2 * 2 * 2       | 否 |

### 流派属性数值
**灼烧**：火球爆炸时，有20%的概率对范围内敌对单位附加**灼烧**效果，持续造成火焰伤害。  

**畏水**：如果火球术经过水源或在露天的下雨场景中时，火球伤害降低25%。
