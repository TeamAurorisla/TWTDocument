# 涌焰

## 基本属性

- 法术类型：攻击
- 法术流派：炽焰
- 魔力需求：10
- 法术稀有度：普通
- 法术强度倍率：0.6
- 法术ID：flame_surge
- 英文：Flame Surge Spell
- 释放类型：召唤
- 可见性：可见

## 描述
涌焰法术会在指定地点召唤一道猛烈的火焰喷泉，对范围内的所有敌人造成一次直接伤害，并在该区域留下火焰。  
涌焰法术的伤害与范围内的敌对生物数量有关，敌对数量越多，伤害越低。

## 数值
### 基础数值
火焰喷泉伤害：*20*  
火焰喷泉影响范围：*5 * 5*（不可以被修正）  
生物数量影响系数：*0.2*（不可以被修正）  

| 属性      | 数值    | 能否修正 |
| ----------- | ----------- |  :---: |
| 火焰喷泉伤害     | 20      | 是 |
| 火焰喷泉影响范围      | 5 * 5       | 否 |
| 生物数量影响系数   | 0.2        | 否 |

### 流派属性数值
**灼烧**：火焰喷泉释放时，有50%的概率对范围内敌对单位附加**灼烧**效果，持续造成火焰伤害。  

**爆燃**：火焰喷泉释放时，有40%的概率对目标添加**爆燃**状态效果。  

**畏水**：在水体方块上释放时，法术直接失效。在露天雨天下释放时，地面火焰无法生成，且直接伤害降低25%。
