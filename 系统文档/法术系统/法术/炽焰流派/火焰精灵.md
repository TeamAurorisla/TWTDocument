# 火焰精灵

## 基本属性

- 法术类型：召唤
- 法术流派：炽焰
- 魔力需求：3
- 法术稀有度：普通
- 法术强度倍率：1.0
- 法术ID：fire_spirit
- 英文：Fire Spirit
- 释放类型：范围
- 可见性：可见

## 描述
火焰精灵法术会在玩家周围召唤三个火焰精灵，火焰精灵会跳向距离玩家最近的敌对目标，并在目标周围造成范围伤害。如果火焰精灵在接触敌对目标前死亡，则该精灵不会产生任何效果。

## 数值

### 基础数值
火焰精灵血量：*1*（不可以被修正）  
火焰精灵伤害：*6*  
火焰精灵伤害范围：*3 * 3*（不可以被修正）  


| 属性      | 数值    | 能否修正 |
| ----------- | ----------- |  :---: |
| 火焰精灵血量     | 1      | 否 |
| 火焰精灵伤害      | 6       | 是 |
| 火焰精灵伤害范围   | 3 * 3        | 否 |

### 流派属性数值

**灼烧**：火焰精灵接触的目标，有50%的概率对周围敌对单位附加**灼烧**效果，持续造成火焰伤害。  

**爆燃**：火焰精灵有30%的概率对目标添加**爆燃**状态效果。

**畏水**：如果火焰精灵接触到水源或在露天的下雨场景中时，火焰精灵立即死亡且不会产生任何效果。  

## 法术精进

- 熟练：火焰精灵血量提升至**4点**，灼烧效果概率提高至**60%**，
    - 精进需求：
- 掌握：火焰精灵伤害提高**2点**，灼烧效果概率提高至**80%**，爆燃状态触发概率提高至**40%**
    - 精进需求：
- 共鸣：火焰精灵伤害提高**2点**，火焰精灵被召唤时，有**40%**的概率会吸引敌对生物的仇恨，灼烧效果概率提高至**90%**，爆燃状态触发概率提高至**50%**
    - 精进需求：
- 谐律：火焰精灵伤害范围修正至**4 * 4**，火焰精灵伤害提高**2点**，灼烧效果概率提高至**100%**，爆燃状态触发概率提高至**60%**
    - 精进需求：
