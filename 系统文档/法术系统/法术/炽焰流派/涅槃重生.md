# 涅槃重生

## 基本属性

- 法术类型：防御
- 法术流派：炽焰
- 魔力需求：10
- 法术稀有度：稀有
- 法术强度倍率：0.0
- 法术ID：blaze_phoenix
- 英文：Blaze Phoenix Spell
- 释放类型：本体
- 可见性：可见

## 描述
涅槃重生法术会给予玩家持续的“凤凰庇护”效果。当玩家的生命值低于 30% 时，凤凰庇护效果会自动触发，给予玩家生命恢复效果。并向前方发射一道凤凰样式的火焰冲击波，对路径上的敌人造成伤害。冲击波每对一个敌人造成伤害时，玩家额外获得生命值。  


## 数值
### 基础数值
凤凰庇护触发生命值阈值：*30%*（不可以被修正）
凤凰庇护持续时间：*300s*（不可以被修正）
冲击波伤害：*10*
冲击波治疗量：*1*
冲击波距离：*12格*（不可以被修正）

| 属性      | 数值    | 能否修正 |
| ----------- | ----------- |  :---: |
| 凤凰庇护触发生命值阈值     | 30%      | 否 |
| 凤凰庇护持续时间      | 300s       | 否 |
| 冲击波伤害   | 10        | 是 |
| 冲击波治疗量   | 1        | 是 |
| 冲击波距离   | 12格        | 否 |

### 流派属性数值
**灼烧**：凤凰冲击波有50%的概率对路径上的敌人附加**灼烧**效果，持续造成火焰伤害。

**爆燃**：凤凰冲击波有30%的概率对路径上的敌人添加**爆燃**状态效果。

**畏水**：当玩家处于露天场景且下雨时，凤凰庇护效果持续时间降低50%，凤凰冲击波伤害降低25%。
