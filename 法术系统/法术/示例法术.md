# 示例法术：惊雷

## 基本属性

- 法术类型：攻击
- 法术流派：雷电
- 魔力需求：28
- 法术稀有度：史诗
- 法术ID：thunder_strike
- 英文：Thunder Strike Spell

## 描述
惊雷法术会释放三道强力雷击，雷击一定会击中索敌范围内血量最高的敌对单位；若命中同一目标，雷击伤害降低25%。  
如果在命中单位周围有其他敌对单位，一个敌对单位受到一次小型闪电攻击

## 数值

### 基础数值
>[!NOTE]
>基础数值一般可以受到法术强度和其他特性的修正，为方便标识，不可以修正的数值需要特别标注

雷击伤害：*50 * 3*  
小型闪电伤害：*8*  
雷击索敌范围：16 * 16（不可以被修正）   
小型闪电索敌范围：5 * 5（不可以被修正）  
雷击间隔时间：0.5s（不可以被修正）  

| 属性      | 数值    | 能否修正 |
| ----------- | ----------- |  :---: |
| 雷击伤害      | 50*3      | 是 |
| 小型闪电伤害   | 8        | 是 |
| 雷击索敌范围      | 16*16       | 否 |
| 小型闪电索敌范围   | 5*5       | 否 |
| 雷击间隔时间   | 0.5s        | 否 |


### 流派效果数值
>[!NOTE]
>流派效果是同一流派共有的效果，但是根据法术的不同，效果也有区别；需要注意的是，流派效果不能被法术强度修正，但应该保留可以修改的接口（可能有法术需要）  

**导电**：对于“机械/傀儡”生物有伤害增益，伤害增加*50%*（例如这里的50%就是有区别的数值，这玩意应该可以用方法...吧）    
**制导**：法术一定会命中索敌范围内最高血量的生物


## 法术精进
>[!NOTE]
>法术升级是在原有数值上修改，升级之间叠加；之后才是其他修正，材料是瞎写的

- 熟练：雷击伤害提高**10%**，小型闪电伤害提高**25%**  
    - 精进需求：4 * 魔力精华（随便起的），1 * 固化雷电  
- 掌握：雷击伤害提高**15%**，小型闪电伤害提高**15%**  
    - 精进需求：16 * 魔力精华，1 * 雷电属性稀有法术符文
- 共鸣：雷击伤害提高**10%**，小型闪电会释放两次  
    - 精进需求：1 * 雷电魔力精华，8 * 浮光锭
- 谐律：雷击伤害提高**10%**，雷击法术索敌范围修改为20 * 20，雷击对生命高于80%的目标造成额外20%伤害
    - 精进需求：4 * 雷电魔力精华，64 * 浮光锭

2. 在“失落文明遗迹-形体”的非核心结构中获得（瞎写的）
