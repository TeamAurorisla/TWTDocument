# 风刃

## 基本属性

- 法术类型：攻击
- 法术流派：疾风
- 魔力需求：8
- 法术稀有度：普通
- 法术强度倍率：0.5
- 法术ID：wind_blade
- 英文：Wind Blade Spell
- 释放类型：投射
- 可见性：可见

## 描述
风刃法术会释放出数道锋利的风刃，风刃会以直线飞行一段距离，并对路径上的敌对单位造成伤害。

## 数值
### 基础数值
风刃伤害：*6*  
风刃飞行距离：*15*（不可以被修正）  
风刃数量：*3*（不可以被修正）  

| 属性      | 数值    | 能否修正 |
| ----------- | ----------- |  :---: |
| 风刃伤害     | 6      | 是 |
| 风刃飞行距离      | 15       | 否 |
| 风刃数量   | 3        | 否 |
